# LicoCode.github.io

## 项目简介
LicoCode.github.io 是一个基于网页的睡眠状态模拟器，结合天气效果，提供用户互动体验。用户可以通过点击或触摸图标来模拟不同的睡眠阶段，并根据地理位置获取天气信息。

## 安装与使用
1. 克隆或下载此项目到本地。
2. 确保您有一个现代浏览器来运行该应用程序。
3. 打开 `index.html` 文件以查看应用程序。

## 功能
- **睡眠状态模拟**: 用户可以通过点击或触摸图标来模拟不同的睡眠阶段（浅睡眠、核心睡眠、深度睡眠、快速眼动）。
- **天气效果**: 根据用户的地理位置获取天气信息，并在页面上显示相应的天气效果（如雨、雪、云等）。
- **动画效果**: 提供流畅的动画效果，增强用户体验。
