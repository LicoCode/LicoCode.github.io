# LicoCode.github.io

## 项目简介
LicoCode.github.io 是一个基于网页的天气模拟器，结合睡眠状态和天气效果，提供用户互动体验。用户可以通过点击或触摸图标来模拟不同的状态，并根据地理位置获取实时天气信息。

## 安装与使用
1. 克隆或下载此项目到本地。
2. 确保您有一个现代浏览器来运行该应用程序。
3. 打开 `index.html` 文件以查看应用程序。

## 功能
- **天气效果**: 根据用户的地理位置获取实时天气信息，并在页面上显示相应的天气效果（如雨、雪、云、闪电等）。
- **昼夜切换**: 根据当前时间动态切换白天和黑夜的背景效果。
- **睡眠状态模拟**: 用户可以通过点击或触摸图标来模拟不同的状态（如睡眠、清醒、愤怒等）。
- **动画效果**: 提供流畅的动画效果，增强用户体验。
- **响应式设计**: 支持不同设备的屏幕尺寸，提供良好的用户体验。

## 天气代码说明
- **天气类型**:
  - 雨：小雨、中雨、大雨（代码范围：51-57, 61, 63, 65, 80-82）
  - 雪：小雪、中雪、大雪（代码范围：71-75, 85-86）
  - 云：晴间多云、雾（代码范围：1-3, 45, 48）
  - 雷暴：轻微、中等、伴有冰雹（代码范围：95, 96, 99）
- **风速等级**:
  - 无风、轻风、中风、大风等，基于风速范围动态调整。
